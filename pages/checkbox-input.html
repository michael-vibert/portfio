Using a Hidden Checkbox Input to Control Your Page Functions

Nov. 3rd 2022

As a beginner HTML and CSS learner, you may have found yourself wondering how your content can become more dynamic, and more functional. You may want to control something like the popular “Switch to Dark Theme” mode or want to display a side-bar menu when you click on an icon, which is super common in mobile pages.

So let’s take a quick and easy stroll into the CSS and HTML that can help you make these cool features happen - Without any Javascript or fancy framework.

We are going to use the humble input type known as a “checkbox”. A checkbox has two states:

  1. Checked
  2. Unchecked

We can link the state of the checkbox with another HTML element through the use of a “label” element type.
