Benford's Law in Python

Sept. 5th 2022

Benford's Law describes the unintuitive truth that not all numbers in a random set of numbers have the same likihood of being present. You'd think that if you had a set of 1000 totally random numbers that you would roughly have 100 that start with the number 1, 100 that start with 2 and so on. But it's not the case.

I decided to investigate this law in a different way. I made a little program in Python that does the following:

  1. Generate lists of "random numbers"
  2. Check if the first number is a "1"
  3. Count how many do start with "1" in each list
  4. Tally the over all count

The point here for me was to test the random number generator supplied in the random import with Python. It does tell you that it is on "pseudo random" so I wasn't expecting to see truly random number selection.

In the code supplied here I'm checking lists of 50 integers between 1 & 1 million for how many start with a "1" And running that 100 times.

It seems to return about 500-550 digits starting with a "1" which is exactly 10-11%.

Please feel free to improve the code, make suggestions or tell me its not doing what I think its doing, that's how we learn :) 

              
    import random


    def randomNum():
      num = random.randrange(0, 1000000)
    return num


    def sortlist(listy):
      count=0
      for x in listy:
        f = str(x)
        if f[0] == "1":
        count+=1
      else:
        continue

      return count


    def listMaker():
      counter=0
      x=0
      list1=[]
      whilex<50:
        num = randomNum()
        list1.append(num)
        x += 1
        counter += (sortlist(list1))
        print("Count=" + str(sortlist(list1)))

      return counter


    def main():
      counter = 0
      i = 0
      while i < 100:
      counter += listMaker()
      i += 1
      print(counter)


    if__name__== "__main__":
      main()
